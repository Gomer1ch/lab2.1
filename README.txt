Завдання 1
Дан список дисциплін, що вивчаються у групі і звітність по них. Скласти 
програму, яка б за назвою дісципліни видавала б звітність по ній.
(Математика (іспит, залік); Іноземна мова (іспит, залік)).
Алгоритм:
1. Створення списку дисциплін
2. Вибір «кейсу» в залежності від введеної дисципліни та одержання 
звітності по ній
